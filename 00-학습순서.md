## 📱 빠른 참고

### 배포 명령어 순서

```bash
# Serverless Framework 배포 (Day 1-2)
cd examples/01-hello-world
npm install
npm run offline              # 로컬 테스트 (선택)
npm run deploy               # AWS 배포

# SAM 배포 (Day 3-4)
cd examples/03-hello-world-sam
npm install
sam build
sam deploy --guided          # 첫 배포 (설정 저장됨)
sam deploy                    # 이후 배포 (설정 재사용)
```
